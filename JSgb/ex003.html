O que é DOM?

DOM é um acrônimo para documment Object Model que é um modelo de objetos para documentos, basicamente é um conjunto de objetos dentro do navegador que vai dar acesso aos componentes internos do website. O DOM está presente quando está rodando JavaScript dentro do navegador.

Árvore dom

Basicamente a árvore DOM começa da raiz e essa raiz dentro do nagevador é chamada de window, então tudo dentro do JavaScript está dentro de um objeto chamado window, a janela do navegador é um objeto DOM. Dentro do "window" tem vários outros objetos.

  • location
    • Esse objeto diz qual é a localização do site, qual é a url,
      qual é a página atual, qual foi a página anterior
  • document
    • Esse objeto diz qual é o documento atual sendo utlizado.
  • history
    • Vai guardando as localizações de movimentações dentro da página, de onde estava anteriormente na pagina e pra onde pode ir.
